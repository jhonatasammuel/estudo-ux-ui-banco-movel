# 📱 Protótipo de App Bancário (Estudo de UX/UI)

Este projeto foi desenvolvido para a disciplina de Design para Dispositivos Móveis. O objetivo foi criar uma interface funcional que une a robustez das operações bancárias tradicionais com a identidade visual moderna das fintechs.

### 🎯 O Desafio
- **Benchmarking**: Análise das funcionalidades do app Itaú e da identidade visual do Nubank.
- **Requisitos Funcionais**: Implementação de fluxos de **Pix** e **Histórico de Transações (Extrato)**.
- **Prototipagem**: Criação de um fluxo interativo no Figma para simular a jornada do usuário.

### 🛠️ Ferramentas Utilizadas
- **Figma**: Design de interface, componentes e prototipagem interativa.
- **Metodologia**: Foco em usabilidade e navegação intuitiva.
